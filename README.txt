# ratio v1.1
Ratio it's an easy jQuery plugin to maintain the aspect ratio of a div depending to the height (or width) of it container.
Ratio is very useful for reponsive design.

Plugin details :

-> Default settings : 
$(element).ratio({
	width : 1,
	height : 1,
	ref : "width"
})

-> Description : 
	width : set the width for your ratio
	height : set the height for your ratio
	ref : set if the block should depend of height or width of it container (width/height)
